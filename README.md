# KSH Safety Dashboard

An interactive Safety & EHS Dashboard built for KSH to visualize and monitor
workplace safety performance.

## Objectives
- Replace static Excel reports with an interactive dashboard
- Track key safety KPIs such as:
  - Accident Free Days
  - Man-Hours
  - LTIFR
  - Accident Rate
  - Training & Observations
- Enable filtering using sliders (Year, Month, Location)

## Tools Used
- Microsoft Excel (Data Source)
- Power BI (Dashboard & Analytics)
- GitHub (Version Control & Documentation)
